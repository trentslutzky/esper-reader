# esper-reader
Curses-based command line audiobook player.

## features
- Save position in audiobook
- Change playback speed
- Forward and backward 15 seconds

##
Not packaged at all yet. To use, clone repo and use:
```sh
$ ./esper_ab <path/to/audiobook>
```
